Recipe Website Tutorials¶

This is a series of tutorials which lay the foundation for using the Bottle web application framework to create a Recipe Website service. Learn more about Bottle at The Bottle Website.

According to the Bottle documentation the following is the simplest Bottle application that you can make:

from bottle import route, run

@route('/hello/:name')
def index(name='World'):
    return '<b>Hello %s!</b>' % name

run(host='localhost', port=8080)

Note

Want to see this in action now? Use the ‘bootstrapbot.py’ script

We’ve bundled all of this into a bootstrap script called bootstrapbot.py, try it out by doing the following:

  1. Download it: http://dl.dropbox.com/u/1004432/bootstrapbot.py

  2. Run it:

    python bootstrapbot.py

  3. Launch it. From the directory where your ran the bootstrap launch the helloworld.py program:

    venv/bin/python helloworld.py

When you visit http://localhost:8080/hello/world in a browser, you will see the text Hello world!.

Having trouble? Are you behind a proxy? Read Dealing with on campus proxies

As our tasks become more complex we will add supporting templates, CSS, JS and image files. All examples will be done on Unix/Linux and have not been thoroughly tested on Windows. If you want to use Windows there are notes available on Installing Bottle and Virtualenv on Windows behind the UWI proxy.

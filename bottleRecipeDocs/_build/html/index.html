Recipe Website Tutorial¶

This tutorial is based on the Bottle web application development microframework. Learn more about Bottle at The Bottle Website.

According to the Bottle documentation the following is the simplest Bottle application that you can make:

from bottle import route, run

@route('/hello/:name')
def index(name='World'):
    return '<b>Hello %s!</b>' % name

run(host='localhost', port=8080)

When saved to helloworld.py, the above application can be run via:

$ pip install bottle
$ python helloworld.py

When you visit http://localhost:8080/hello/world in a browser, you will see the text Hello world!.

This is a simple single file application. As our tasks become more complex there comes a point when we will need supporting templates, CSS, JS and image files. This tutorial will focus on building a “Recipe Website” using the Bottle microframework.
